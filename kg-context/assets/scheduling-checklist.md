# KG Scheduling Checklist — Per Location

Send this to every new account after signing. One block per location.

- What time can we start cleaning?
- When do we need to be out?
- Can we keep a key on file for access? Can we pick this up at the first cleaning or does it need to be picked up prior?
- Who is the best scheduling / on-site contact for this location? (name and phone or email)
- Do we need to confirm dates with you prior to each cleaning, or can we schedule and simply let you know when we're coming?
- Is there an alarm code?
- Any special notes for this location?
